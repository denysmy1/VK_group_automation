1) We create an application in VK.
1.1 We create a stendelone application for VK.
1.2 We get a service access key for it according to the instructions https://vk.com/dev/access_token?f=3.%20Сервисный%20ключ%20доступа

2) Connecting publics
2.1 In the above instructions, we get an access token for a community with administrative permission

3) Adding publics to the database (DB)
3.1 Adding publics to the database using the newGroupInBD.py script

4) Run the script

4.1 To run the script, write "pip install vk_api" in the terminal of your OS, this will download the vk api library to your device

ATTENTION!!! The script requires Python3
